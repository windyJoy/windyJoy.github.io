基本概念#

堆是一种特殊的完全二叉树（除了最后一层，其他层都是满的，且最后一层节点靠左排列）。其每个节点的值都满足与子节点的特定大小关系。因此堆分为最大堆和最小堆。在C++中，堆的元素存储在数组中，并利用索引关系表示树结构。对于任意节点在数组中的位置 i，其父节点的位置为(i-1)/2，左子节点的位置为2*i+1，右子节点的位置为2*i+2。C++的标准库提供了priority_queue容器实现堆和<algorithm>中的堆算法堆化数组。

优先队列#

创建最大堆：

priority_queue<int> max_heap;
priority_queue<int,vector<int>,less<int>> max_heap;

创建最小堆：

priority_queue<int,vector<int>,greater<int>> min_heap;

C++默认是使用升序排列，所以最大堆有简略的写法。但是最小堆就得显示表明。

初始化堆的两种方式：

priority_queue<int> max_heap;
for(int x:nums) max_heap.push(x);

priority_queue<int,vector<int>,greater<int>> min_heap(nums.begin(),nums.end());

当堆存储的元素不为单一的int,为其他类型时，例如tuple元组：

priority_queue<tuple<int,int>,vector<tuple<int,int>>,greater<>> minHeap;

//有三种方式:

// 1. emplace
minHeap.emplace(1, 2);  // 直接构造，最高效
// 2. push + make_tuple
minHeap.push(make_tuple(1, 2));  // 使用辅助函数
// 3. push + 显式构造
minHeap.push(tuple<int, int>(1, 2));  // 显式指定类型

当最小堆使用元组当元素时，其比较规则为：

  1. 比较第一个元素
  2. 如果相等，比较第二个元素
  3. 如果还相等，比较第三个元素
  4. 以此类推

堆化数组#

#include<vector>
#include<algorithm>
vector<int> nums = {3,6,5,1,2}

//vector堆化
make_heap(nums.begin(),nums.end())
for(int x:nums) cout << x;

//随机访问并修改元素
nums[3]=9;
make_heap(nums.begin(),nums.end())

//vector末尾添加元素
nums.push_back(9)
push_heap(nums.begin(),nums.end())

//弹出元素
pop_heap(nums.begin(),nums.end())
nums.pop_back()

优先队列不可以随机访问其中的元素，只能访问堆的头元素，vector支持随机访问，vector堆化在保留堆数据结构的基础上支持随机访问，但是需要手动维护堆的数据结构。

Top K 问题#

拿到一个数组，想要找到其第k大的数字，有两种方式：

  1. 对数组排序。
  2. 维护一个大小为k的最小堆，遍历整个数组。
方面 堆方法 排序方法
时间复杂度 $O(nlogk)$ $O(nlogn)$
空间复杂度 $O(k)$ $O(n)$
适用场景 $k«n$ 需要完全排序

堆的O(log n)插入删除和O(1)获取极值的特性，使其在某些场景中非常高效
